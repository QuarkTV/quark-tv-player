## Quick Start

After [installing the package](/docs/installation), you can start using QuarkTV Player in your application.

### 1. Initialize the BunnyStream Client

First, create an instance of `BunnyStreamClient` with your Bunny.net credentials:

```jsx
import { BunnyStreamClient } from '@quarktv/quark-tv-player'

const client = new BunnyStreamClient({
  apiKey: 'your-api-key',
  libraryId: 'your-library-id'
})
```

### 2. Use the Player Component

Now you can use the `QuarkTVPlayer` component in your application:

```jsx
import { QuarkTVPlayer } from '@quarktv/quark-tv-player'

export default function VideoPlayer() {
  return (
    <QuarkTVPlayer
      videoId="your-video-id"
      config={client}
      streamerInfo={{
        name: "VanixQC",
        title: "Just Chatting",
        category: "IRL",
        viewers: 1234,
        avatarUrl: "https://example.com/avatar.jpg"
      }}
    />
  )
}
```

### 3. Customize the Player

You can customize the player's appearance using CSS variables:

```css
media-player {
  --media-brand: #9147ff; /* Primary color */
  --media-focus-ring: #9147ff; /* Focus ring color */
  --media-background: #0e0e10; /* Player background */
}
```

## Component Props

### Required Props

- `videoId`: The ID of the video from Bunny.net
- `config`: An instance of `BunnyStreamClient`

### Optional Props

- `streamerInfo`: Information about the streamer and stream
  ```typescript
  interface StreamerInfo {
    name: string;
    title?: string;
    category?: string;
    viewers?: number;
    avatarUrl?: string;
  }
  ```

## Error Handling

The player includes built-in error handling and will display appropriate messages for common issues:

```jsx
<QuarkTVPlayer
  videoId="your-video-id"
  config={client}
  onError={(error) => {
    console.error('Video playback error:', error)
  }}
/>
```
