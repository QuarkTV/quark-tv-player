## Contributing

We welcome contributions from the community! Here's how you can help:

1. **Bug Reports**: Open an issue on GitHub if you find a bug
2. **Feature Requests**: Share your ideas for new features
3. **Code Contributions**: Submit pull requests for bug fixes or new features
4. **Documentation**: Help improve our documentation

### Getting Started with Development

```bash
# Clone the repository
git clone https://github.com/quarktv/quark-tv-player.git

# Install dependencies
npm install

# Start development server
npm run dev
```

For more detailed information about contributing, please check our [GitHub repository](https://github.com/quarktv/quark-tv-player).
